# Genome Sets

Place local FASTA and annotation files here.

Example:

```text
genome_sets/
  MyGenome/
    genome.fna
    genes.gff3
    proteins.faa
```

Then add optional display metadata to `genomes.csv`.

---

The `TAIR10_blastsets` directory is supported by this app. On Linux, you can download it with:
```bash
wget -r https://www.arabidopsis.org/download_files/Sequences/TAIR10_blastsets/
```
